# Equalizer Pro

## Description

The best unseen equalizer out there. Take it with a pinch of salt!

## Features

-   Different filters
-   Gain knob
-   Option to widen the filter

## Filters Used

-   Lowpass filter
-   Highpass filter
-   Bandpass filter
-   Bandstop filter
-   Chebyshev filter
-   Butterworth filter

Dr. Dre - Still (Instrumental)
